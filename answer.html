The Array.map() method allows to iterate over an array and modify its elements using a callback function.
Examples:
const elements = array.map(element =>element*2)

array.forEach method doesn't return an array. It only uses when we don't need to return anything but only operation on elements
Examples
array.forEach(element=> console.log(element))

Array filter method returns an array who will pass the conditions.
Examples
const passed = array.filter(element=> element < 25)

Array.find almost similar to array.filter method but it returns only one variable. The first variable that passes the condition. Examples const variable = array.find(element=> element > 10)
